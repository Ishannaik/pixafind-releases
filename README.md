# Local AI Search

Release artifacts for Local AI Search — privacy-first AI file search for your PC.

## About This Repo

This repository hosts installer downloads and update artifacts for [Local AI Search](https://github.com/Ishannaik/local-ai-search). The app uses Tauri's built-in updater to check this repo for new releases.

> The source code lives in the [main repository](https://github.com/Ishannaik/local-ai-search) (private).

## Download

| Platform | Installer | Status |
|----------|-----------|--------|
| Windows  | `.msi` / `.exe` (NSIS) | Available |

Check the [Releases](https://github.com/Ishannaik/local-ai-search-releases/releases) page for the latest version.

## Features

- **AI Image Search** — find photos by describing them in natural language (powered by SigLIP2)
- **Text Search** — hybrid BM25 + semantic search across documents
- **Find Similar** — select an image and discover visually similar ones (powered by DINOv3)
- **Metadata Search** — filter by date, camera, location, and file properties
- **100% Local** — zero cloud calls, zero telemetry, everything runs on your machine

## Requirements

- Windows 10/11 (64-bit)
- 4 GB RAM minimum, 8 GB recommended
- ~500 MB disk space (app + models)
